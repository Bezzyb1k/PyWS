# Числа в Python

В Python есть два основных числовых типа
- int (целые)
- float (вещественные)

# Основные операции
- '+' - сложение
- '-' - вычитание
- '*' - умножение
- '/' - деление (всегда возвращает float)
- '//' - деление без остатка (всегда возвращает int)
- '%' - остаток от деления
- '**' - возведение в степень

# Порядок выполнения операций
Как и в математике приоритет выполнения является таким:
скобки --> степень --> умножение/деление --> сложение/вычитание

# Сравнение чисел
- '==' - равно
- '!=' - не равно
- '>' - больше
- '<' - меньше
- '>=' - больше или равно
- '<=' - меньше или равно

# Встроенные функции
- abs() - модуль числа
- round() - округление числ
- pow() - возведение в степень
